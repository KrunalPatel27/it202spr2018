Answers to questions

  1. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
  2. A "block-level" element occupies the entire space of its parent element. They always start on a new line and takes up the full width of parent element.
  3. div is by default a block element.
  4. An inline element does not start on a new line and only takes up as much width as necessary. Also they are bounded to their parents.
  5. span is an example element that is by default rendered as inline element.
  6. CSS can be linked through another file or embedded inside html.
  7. In CSS, selectors are patterns used to select the element(s) you want to style.
  8. The following CSS rule can be used to set the background color of body to #0000FF (blue).body { background-color: #0000FF; }
